# 基于图的测试框架
## 前端
&emsp;&emsp;使用vue3、element-plus、typescript、以及Drawflow编写。支持json导入导出。
## 后端
&emsp;&emsp;使用python3，接收测试图json序列化数据，从BeginNode，按照流程分别运行各个测试节点，并最终到达EndNode，或告知失败。
